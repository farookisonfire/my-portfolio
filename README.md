### Key Technologies
* HTML 5
* CSS3
* Bootstrap
* JavaScript ES6
* React
* Node.js
* Express

### Development Launch Instructions:
* Install the`npm` package manager and `node.js` runtime environment.
* Launch the application and start server:
```bash
λ npm install
λ npm run serve
λ node .
```
